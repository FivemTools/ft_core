# ft_core
To create resources on FiveM

### Prerequisites

Nothing

### Installing

```
start ft_core
```

or put in to dependencies in __resource.lua :

```
dependency 'ft_core'

or

dependencies {
  ...
  'ft_core',
  ...
}

```
